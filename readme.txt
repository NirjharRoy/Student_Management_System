extract the Group_2_CompactDistributedObjects.zip

,you will get 2 folders.

"Executables" contain all executable jars
"SourceCodes" contain all source codes.

To execute project

a) go to "Executables/Server". Open terminal  and make this directory as the current directory  and run the following command "./startserver.sh"

b) then go to "Executables/Client".Open terminal  and make this directory as the current directory  and run the following command "./runClient.sh"
